# SafeLens IA

## Sobre o projeto
SafeLens IA é uma solução desenvolvida para o evento Grand Prix com o objetivo de garantir acesso seguro a sistemas de inteligência artificial por meio de dispositivos vestíveis.

## Problema
Dispositivos como óculos inteligentes podem acessar dados sensíveis, como imagem, áudio e localização, o que aumenta os riscos de vazamento de informações e acesso indevido.

## Solução
A solução propõe uma arquitetura segura que controla o acesso à inteligência artificial, garantindo que apenas usuários autenticados possam interagir com o sistema e que dados sensíveis sejam protegidos.

## Funcionalidades
- Autenticação de usuário com geração de token
- Controle de acesso às requisições
- Filtro de dados sensíveis
- Integração com inteligência artificial
- Registro de atividades (logs)

## Tecnologias utilizadas
- Node.js
- Express
- JSON Web Token (JWT)
- API de inteligência artificial
- HTML, CSS e JavaScript

## Como executar

### Backend
1. Acesse a pasta backend
2. Instale as dependências:
   npm install
3. Inicie o servidor:
   node server.js

### Frontend
Abra o arquivo index.html no navegador
